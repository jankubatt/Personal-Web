Smart Home Project

The goal of this project is to make my room smart as much as possible. So that includes smart lighting, thermostat, and other stuff. This project is in progress. For now, I have done an LED strip behind my table and temperature + humidity sensor for my room. Sensors and lights are connected to Wemos (Arduino) from there, the data is sent to RPi 3 where my Home Assistant is located. In Home Assistant I use add-on called ESPHome.




Parts

  • Raspberry Pi 3
  • Wemos D1 R2 (Arduino + ESP8266)
  • DHT 11 (Temp & Hum sensor)
  • LED strip
